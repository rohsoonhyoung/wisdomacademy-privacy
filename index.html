1. 개인정보 처리방침 개요

지혜학당(이하 "본 앱")은 아이들의 언어 학습을 돕는 교육용 게임 앱입니다. 본 개인정보 처리방침은 본 앱이 수집, 사용, 보호하는 개인정보에 관한 사항을 설명합니다.

2. 수집하는 개인정보의 항목

2.1 필수 수집 정보

  • 기기 정보 (기기 모델, OS 버전, 광고 ID)
  • 게임 진행 데이터 (점수, 레벨, 달성도)
  • 앱 사용 통계 (게임 플레이 시간, 기능 사용 현황)

2.2 선택 수집 정보

  • 닉네임 (사용자가 직접 설정)
  • 랭킹 데이터 (순위표 참여 시)

3. 개인정보의 처리 목적

  • 게임 진행상황 저장 및 복원
  • 개인 맞춤형 학습 콘텐츠 제공
  • 랭킹 시스템 운영
  • 앱 사용 패턴 분석을 통한 서비스 개선
  • 맞춤형 광고 제공
